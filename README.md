# 📊 Investimentos para Iniciantes com NotebookLM

## 📌 Contexto
Este projeto foi desenvolvido para o desafio da DIO utilizando o NotebookLM como ferramenta de aprendizagem ativa sobre educação financeira.

## 🎯 Objetivos de Estudo
- Entender o que são investimentos
- Diferenciar renda fixa e variável
- Compreender reserva de emergência
- Explorar risco, liquidez e rentabilidade

## 📚 Fontes Utilizadas
- CVM - Portal do Investidor
- Banco Central do Brasil - Cidadania Financeira
- B3 Educação
- ANBIMA Educação
- XP Investimentos (guia para iniciantes)
